一、实验目的

1. 理解和掌握产生式知识表示方法。

2. 能够利用产生式知识表示方法进行正向推理和反向推理。

3. 能够用Python语言建立产生式系统的规则库。

 

二、实验背景

建立一个动物识别系统的规则库，用以识别虎、豹、斑马、长颈鹿、企鹅、鸵鸟、信天翁等7种动物。

为了识别这些动物，可以根据动物识别的特征，建立包含下述规则的规则库(总共15条)：

R1：if动物有毛发then动物是哺乳动物。

R2：if动物有奶then动物是哺乳动物。

R3：if动物有羽毛then动物是鸟。

R4：if动物会飞and会生蛋then动物是鸟。

R5：if动物吃肉then动物是食肉动物。

R6：if动物有犀利牙齿and有爪and眼向前方then动物是食肉动物。

R7：if动物是哺乳动物and有蹄then动物是有蹄类动物。

R8：if动物是哺乳动物and反刍then动物是有蹄类动物。

R9：if动物是哺乳动物and是食肉动物and有黄褐色and有暗斑点then动物是豹。

R10：if动物是哺乳动物and是食肉动物and有黄褐色and有黑色条纹then动物是虎。

R11：if动物是有蹄类动物and有长脖子and有长腿and有暗斑点then动物是长颈鹿。

R12：if动物是有蹄类动物and有黑色条纹then动物是斑马。

R13：if动物是鸟and不会飞and有长脖子and有长腿and有黑白二色then动物是鸵鸟。

R14：if动物是鸟and不会飞and会游泳and有黑白二色then动物是企鹅。

R15：if动物是鸟and善飞then动物是信天翁。

如果动物有暗斑点、有长脖子、有长腿、有奶、有蹄，请推理出是什么动物。

 

三、实验代码

# 规则库

rules = [

    ("有毛发", "是哺乳动物"),

    ("有奶", "是哺乳动物"),

    ("有羽毛", "是鸟"),

    ("会飞", "会生蛋", "是鸟"),

    ("吃肉", "是食肉动物"),

    ("有犀利牙齿", "有爪", "眼向前方", "是食肉动物"),

    ("是哺乳动物", "有蹄", "是有蹄类动物"),

    ("是哺乳动物", "反刍", "是有蹄类动物"),

    ("是哺乳动物", "是食肉动物", "有黄褐色", "有暗斑点", "是豹"),

    ("是哺乳动物", "是食肉动物", "有黄褐色", "有黑色条纹", "是虎"),

    ("是有蹄类动物", "有长脖子", "有长腿", "有暗斑点", "是长颈鹿"),

    ("是有蹄类动物", "有黑色条纹", "是斑马"),

    ("是鸟", "不会飞", "有长脖子", "有长腿", "有黑白二色", "是鸵鸟"),

    ("是鸟", "不会飞", "会游泳", "有黑白二色", "是企鹅"),

    ("是鸟", "善飞", "是信天翁")

]

 

# 事实库

facts = set()

 

# 动物特征输入

def input_features():

    features = []

    feature_input = input("请输入动物特征（空格分隔，输入'end'结束）: ")

    while feature_input.lower() != 'end':

        features.append(feature_input)

        feature_input = input("请继续输入动物特征（空格分隔，输入'end'结束）: ")

    return features

 

# 将输入的特征添加到事实库

def add_features_to_facts(features):

    for feature in features:

        facts.add(feature)

 

# 匹配规则

def match_rule(rule):

    return set(rule[:len(rule)-1]).issubset(facts)

 

# 应用规则

def apply_rule(rule):

    facts.add(rule[-1])

 

# 识别动物

def identify_animal():

    while True:

        for rule in rules:

            if match_rule(rule):

                apply_rule(rule)

                if rule[-1] in ["是豹", "是虎", "是长颈鹿", "是斑马", "是鸵鸟", "是企鹅", "是信天翁"]:

                    return rule[-1].split('是')[1]

        break

    return "未知动物"


# 主程序

def main():

    features = input_features()

    add_features_to_facts(features)

    animal = identify_animal()

    print(f"识别的动物是：{animal}")

 

if __name__ == "__main__":

    main()
